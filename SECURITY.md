# # Security Policy

## Supported Versions
We only provide security updates for the latest version of GHOSTNOTE AI.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

At GHOSTNOTE AI, we take security seriously. If you find a vulnerability, please help us by reporting it responsibly.

**DO NOT** open a public Github issue for security bugs.

Instead, email us at: **security@ghostnoteai.com**

Include:
1.  Description of the vulnerability
2.  Steps to reproduce
3.  Potential impact
4.  Your contact info

We will respond within 48 hours. We will work with you to understand and fix the issue.

## What we protect
1.  **User Payments**: All payments go through Stripe. We never store card numbers.
2.  **User Uploads**: Beats and vocals are deleted from our servers after 24 hours unless you save them.
3.  **Accounts**: Passwords are hashed. We use 2FA.

## Bug Bounty
Right now we don’t have a paid bug bounty. But we will credit you in the README for valid reports.

Thank you for helping keep GHOSTNOTE AI safe.Security Policy
